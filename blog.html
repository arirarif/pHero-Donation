What is the difference between innerText and innerHTML in DOM manipulation?

innerText retrieves or sets the visible text within an element, while innerHTML retrieves or sets the HTML content (including tags) inside the element. innerHTML can include both the text and the HTML structure within the element.

How can you dynamically add an element to the DOM?

You can create a new element using document.createElement(), set its attributes or content, and then append it to a parent node using methods like appendChild() or insertBefore().

What is event delegation in the context of the DOM,and why it is useful?

the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated

How do you manipulate an elements attributes and styles using the DOM?

If you want to change other attributes like id , href , or src , you can use the setAttribute() method.
